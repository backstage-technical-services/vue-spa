# Vue SPA

## Technologies
* TypeScript 3.x
* Vue.js 2.x
* Vuex
* Vue Router
* Axios
* FontAwesome 5
* Bootstrap 4

## Commands
* Install dependencies:
    ```sh
    $ yarn install
    ```
* Boot development server with hot reloading 
    ```sh
    $ yarn run serve
    ```
* Lint and fix
    ```sh
    $ yarn run lint --fix
    ```
* Unit tests
    ```sh
    $ yarn run test:unit
    ```
* End to end tests
    ```sh
    $ yarn run test:e2e
    ```
